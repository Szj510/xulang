# Bundled sample media

These six JPEG files are synthetic sample media created for Xulang with OpenAI's built-in image generation on 2026-07-20. They are not user photos and are included only to make the first-run exhibition usable without importing personal media. The lossless source PNGs were converted to quality-88 JPEGs on 2026-07-22 to reduce the Android package size without cropping or changing the composition.
