# Security Policy / 安全政策

## Supported versions / 支持版本

Security fixes are provided for the latest release only.

安全修复只面向最新正式版本。

| Version | Supported |
| --- | --- |
| 1.1.x | Yes |
| 1.0.x and earlier | No |

## Reporting a vulnerability / 报告漏洞

Do not open a public issue, discussion, or pull request for a suspected vulnerability.

请勿通过公开 Issue、Discussion 或 Pull Request 披露疑似漏洞。

Use [GitHub private vulnerability reporting](https://github.com/Szj510/xulang/security/advisories/new). Include:

- the affected version and Android version;
- impact and a minimal reproduction;
- relevant logs with personal data removed;
- whether the issue is already public or actively exploited.

请使用 [GitHub 私密漏洞报告](https://github.com/Szj510/xulang/security/advisories/new)，并提供受影响版本、Android 版本、影响范围、最小复现、已脱敏日志，以及漏洞是否已经公开或遭到利用。

The maintainer will acknowledge a valid report as soon as practical, coordinate a fix privately, and credit the reporter unless anonymity is requested. Please do not test against other people's devices or data.

维护者会尽快确认有效报告并私下协调修复；除非报告者要求匿名，发布说明会给予致谢。请勿使用他人的设备或数据进行测试。
